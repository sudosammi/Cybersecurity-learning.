# 🛡️ CyberSecurity-Beginner Journey

Welcome to my Beginner Cybersecurity and Ethical Hacking repository! 🚀
This repository documents my initial 30+ days of learning Python for cybersecurity, building custom hacking tools, and understanding network fundamentals.

## 📂 Repository Structure

Maine apne saare tools aur scripts ko 3 main modules mein organize kiya hai:

### 1. `01_Python_Fundamentals` (Day 6 - Day 11)
Yahan maine Python ki basics seekhi aur chhote tools banaye.
* 🐍 Loops, Dictionaries, aur Functions ki practice.
* 🧮 Custom Calculators aur File handling scripts.

### 2. `02_Recon_and_Scanning` (Day 12 - Day 20)
Information gathering aur target scanning ke tools.
* 🔍 **Port Scanners:** Target IPs par open ports dhoondhna.
* 🚩 **Banner Grabbing:** Services aur unke versions nikalna.
* 📖 **Wordlist Handling:** Brute-force attacks ke liye custom wordlists banana.

### 3. `03_Network_Sniffing_MITM` (Day 21 - Day 32)
Real network hacking aur Man-In-The-Middle (MITM) attacks.
* 🕵️ **Packet Sniffers:** Network traffic ko intercept aur decode karna.
* 🎭 **ARP Spoofer:** Network mein khud ko router banakar traffic redirect karna.
* 🌐 **DNS Spoofing:** Target ko fake websites par bhejna.
